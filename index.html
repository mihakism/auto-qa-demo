AI 분석

Claude Vision이
화면을 읽습니다

규칙으로 잡을 수 없는 UX 품질 문제를 Claude가 직접 화면을 보고 분석합니다. API 키만 입력하면 활성화됩니다.

✓ 상태 커버리지 (Loading / Empty / Error)
✓ 텍스트 극한 처리 및 말줄임 기준
✓ Lorem ipsum · 플레이스홀더 잔존
✓ 레이아웃 정렬 이상
✓ 접근성 우려 요소 (텍스트·버튼 크기)
✦ AI 분석 완료 — 3건 발견
✅ 1. 상태 커버리지 — 이상 없음. Loading, Empty 상태가 확인됩니다.
⚠️ 2. 텍스트 극한 처리 — 상품명 영역에 말줄임 처리가 없습니다. 긴 텍스트 시 레이아웃 깨짐이 우려됩니다.
✅ 3. 이미지 Fallback — 이미지 영역에 placeholder가 확인됩니다.
⚠️ 4. 카피 완성도 — '홍길동', '상품명 입력' 등 임시 카피가 3곳 남아있습니다.
✅ 5. 레이아웃 일관성 — 전반적으로 정렬이 균일합니다.
⚠️ 6. 접근성 우려 — 하단 탭바 아이콘이 18px로 작습니다. 최소 24px 권장.
